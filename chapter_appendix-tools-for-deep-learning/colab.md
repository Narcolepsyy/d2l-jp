# Google Colab を使う
:label:`sec_colab`

この本を AWS 上で実行する方法は :numref:`sec_sagemaker` と :numref:`sec_aws` で紹介しました。もう一つの選択肢として、Google アカウントを持っていれば、この本を [Google Colab](https://colab.research.google.com/) 上で実行できます。

Colab である節のコードを実行するには、:numref:`fig_colab` に示すように `Colab` ボタンをクリックするだけです。 

![Colab 上である節のコードを実行する。](../img/colab.png)
:width:`300px`
:label:`fig_colab`


コードセルを初めて実行する場合は、:numref:`fig_colab2` に示すような警告メッセージが表示されます。
それを無視するには、"RUN ANYWAY" をクリックしてください。

![ "RUN ANYWAY" をクリックして警告メッセージを無視する。](../img/colab-2.png)
:width:`300px`
:label:`fig_colab2`

次に、Colab はこの節のコードを実行するためのインスタンスに接続します。
具体的には、GPU が必要な場合、
Colab は自動的に GPU インスタンスへの接続を要求します。


## 要約

* Google Colab を使って、この本の各節のコードを実行できます。
* この本のいずれかの節で GPU が必要な場合、Colab は GPU インスタンスへの接続を要求します。


## 演習

1. Google Colab を使って、この本の任意の節を開いてみましょう。
1. Google Colab を使って、GPU を必要とする任意の節を編集して実行してみましょう。
