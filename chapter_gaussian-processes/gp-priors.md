# ガウス過程の事前分布

ガウス過程（GP）を理解することは、モデル構築と汎化について推論するうえで重要であり、能動学習や深層学習におけるハイパーパラメータ調整を含む、さまざまな応用で最先端の性能を達成するためにも重要です。GP は至るところにあり、それが何であり、どのように使えるのかを知っておくことは私たちにとって有益です。

この節では、関数に対するガウス過程の _事前分布_ を導入します。次のノートブックでは、これらの事前分布を使って _事後推論_ を行い、予測を作る方法を示します。次の節は「GP の要点」と見なすことができ、実際にガウス過程を適用するために必要なことを手早く示します。

## 定義

ガウス過程は、_任意の有限個を取ると同時ガウス分布に従う確率変数の集合_ として定義されます。関数 $f(x)$ が、_平均関数_ $m(x)$ と _共分散関数_ または _カーネル_ $k(x,x')$ をもつガウス過程であるとき、$f(x) \sim \mathcal{GP}(m, k)$ と書きます。このとき、任意の入力点 $x$（時刻、空間位置、画像の画素など）の集合で問い合わせた関数値の集合は、平均ベクトル $\mu$ と共分散行列 $K$ をもつ同時多変量ガウス分布に従います。すなわち、$f(x_1),\dots,f(x_n) \sim \mathcal{N}(\mu, K)$ であり、ここで $\mu_i = E[f(x_i)] = m(x_i)$、$K_{ij} = \textrm{Cov}(f(x_i),f(x_j)) = k(x_i,x_j)$ です。

この定義は抽象的でとっつきにくく見えるかもしれませんが、ガウス過程は実際には非常に単純な対象です。任意の関数

$$f(x) = w^{\top} \phi(x) = \langle w, \phi(x) \rangle,$$ :eqlabel:`eq_gp-function`

は、$w$ がガウス（正規）分布から引かれ、$\phi$ が任意の基底関数ベクトル、たとえば $\phi(x) = (1, x, x^2, ..., x^d)^{\top}$ であるとき、ガウス過程です。さらに、任意のガウス過程 $f(x)$ は式 :eqref:`eq_gp-function` の形で表せます。まずはガウス過程に少しずつ親しむために、いくつか具体例を見ていきましょう。そのあとで、それらが実際にはどれほど単純で有用かを理解できるはずです。

## 単純なガウス過程

$f(x) = w_0 + w_1 x$ とし、$w_0, w_1 \sim \mathcal{N}(0,1)$、さらに $w_0, w_1, x$ はすべて 1 次元であるとします。この関数は、内積 $f(x) = (w_0, w_1)(1, x)^{\top}$ と等価に書けます。上の :eqref:`eq_gp-function` では、$w = (w_0, w_1)^{\top}$、$\phi(x) = (1,x)^{\top}$ です。

任意の $x$ に対して、$f(x)$ は 2 つのガウス確率変数の和です。ガウス分布は加法に関して閉じているので、$f(x)$ も任意の $x$ に対してガウス確率変数です。実際、任意の特定の $x$ について $f(x)$ は $\mathcal{N}(0,1+x^2)$ に従うことが計算できます。同様に、任意の入力 $x_1,\dots,x_n$ に対する関数値の集合 $(f(x_1),\dots,f(x_n))$ の同時分布は、多変量ガウス分布です。したがって $f(x)$ はガウス過程です。

要するに、$f(x)$ は _ランダム関数_、あるいは _関数の分布_ です。$w_0, w_1$ の値を繰り返しサンプルし、それに対応する関数 $f(x)$ を可視化することで、この分布についていくつかの洞察を得られます。これらは傾きと切片が異なる直線です。以下のようになります。

```{.python .input}
def lin_func(x, n_sample):
    preds = np.zeros((n_sample, x.shape[0]))
    for ii in range(n_sample):
        w = np.random.normal(0, 1, 2)
        y = w[0] + w[1] * x
        preds[ii, :] = y
    return preds

x_points = np.linspace(-5, 5, 50)
outs = lin_func(x_points, 10)
lw_bd = -2 * np.sqrt((1 + x_points ** 2))
up_bd = 2 * np.sqrt((1 + x_points ** 2))

d2l.plt.fill_between(x_points, lw_bd, up_bd, alpha=0.25)
d2l.plt.plot(x_points, np.zeros(len(x_points)), linewidth=4, color='black')
d2l.plt.plot(x_points, outs.T)
d2l.plt.xlabel("x", fontsize=20)
d2l.plt.ylabel("f(x)", fontsize=20)
d2l.plt.show()
```

もし $w_0$ と $w_1$ を $\mathcal{N}(0,\alpha^2)$ から引くとしたら、$\alpha$ を変えることが関数分布にどのような影響を与えると考えますか？

## 重み空間から関数空間へ

上の図では、モデルのパラメータ分布がどのように関数分布を誘導するかを見ました。私たちはしばしば、モデル化したい関数が滑らかか、周期的か、急激に変化するかなどについては考えを持っていますが、主として解釈しにくいパラメータについて推論するのは比較的面倒です。幸い、ガウス過程は関数について _直接_ 推論するための簡単な仕組みを提供します。ガウス分布は第 1・第 2 モーメント、すなわち平均と共分散行列だけで完全に定まるので、ガウス過程も同様に平均関数と共分散関数で定まります。

上の例では、平均関数は

$$m(x) = E[f(x)] = E[w_0 + w_1x] = E[w_0] + E[w_1]x = 0+0 = 0.$$

同様に、共分散関数は

$$k(x,x') = \textrm{Cov}(f(x),f(x')) = E[f(x)f(x')]-E[f(x)]E[f(x')] = E[w_0^2 + w_0w_1x' + w_1w_0x + w_1^2xx'] = 1 + xx'.$$

これで、パラメータ分布からサンプルする必要なく、関数分布を直接指定してサンプルできるようになりました。たとえば $f(x)$ から引くには、問い合わせたい任意の $x$ の集合に対応する多変量ガウス分布を作り、そこから直接サンプルすればよいのです。この定式化がどれほど有利か、これから見えてくるでしょう。

まず、上の単純な直線モデルに対するほぼ同じ導出が、$w \sim \mathcal{N}(u,S)$ をもつ任意の形 $f(x) = w^{\top} \phi(x)$ のモデルに対して、平均関数と共分散関数を求めるために適用できることに注意します。この場合、平均関数は $m(x) = u^{\top}\phi(x)$、共分散関数は $k(x,x') = \phi(x)^{\top}S\phi(x')$ です。$\phi(x)$ は任意の非線形基底関数ベクトルを表せるので、私たちは非常に一般的なモデルクラスを扱っており、_無限個_ のパラメータをもつモデルさえ含まれます。

## 放射基底関数（RBF）カーネル

_放射基底関数_（RBF）カーネルは、ガウス過程および一般のカーネル法において最もよく使われる共分散関数です。
このカーネルは $k_{\textrm{RBF}}(x,x') = a^2\exp\left(-\frac{1}{2\ell^2}||x-x'||^2\right)$ の形をしており、$a$ は振幅パラメータ、$\ell$ は _長さ尺度_ ハイパーパラメータです。

重み空間から出発してこのカーネルを導出してみましょう。次の関数を考えます。

$$f(x) = \sum_{i=1}^J w_i \phi_i(x), w_i  \sim \mathcal{N}\left(0,\frac{\sigma^2}{J}\right), \phi_i(x) = \exp\left(-\frac{(x-c_i)^2}{2\ell^2 }\right).$$

$f(x)$ は、幅 $\ell$ をもち、点 $c_i$ を中心とする放射基底関数の和であり、次の図に示すようになります。


$f(x)$ は $w = (w_1,\dots,w_J)^{\top}$、$\phi(x)$ は各放射基底関数を含むベクトルとして、$w^{\top} \phi(x)$ の形をしているとみなせます。このガウス過程の共分散関数は

$$k(x,x') = \frac{\sigma^2}{J} \sum_{i=1}^{J} \phi_i(x)\phi_i(x').$$

では、パラメータ（および基底関数）の数を無限大にすると何が起こるかを考えましょう。$c_J = \log J$、$c_1 = -\log J$、$c_{i+1}-c_{i} = \Delta c = 2\frac{\log J}{J}$ とし、$J \to \infty$ とします。すると共分散関数はリーマン和になり、

$$k(x,x') = \lim_{J \to \infty} \frac{\sigma^2}{J} \sum_{i=1}^{J} \phi_i(x)\phi_i(x') = \int_{c_0}^{c_\infty} \phi_c(x)\phi_c(x') dc.$$

$c_0 = -\infty$、$c_\infty = \infty$ とすると、無限個の基底関数を実数直線全体に広げ、それぞれを $\Delta c \to 0$ の間隔で配置することになります。

$$k(x,x') = \int_{-\infty}^{\infty} \exp(-\frac{(x-c)^2}{2\ell^2}) \exp(-\frac{(x'-c)^2}{2\ell^2 }) dc = \sqrt{\pi}\ell \sigma^2 \exp(-\frac{(x-x')^2}{2(\sqrt{2} \ell)^2}) \propto k_{\textrm{RBF}}(x,x').$$

ここで少し立ち止まって、何をしたのかを噛みしめる価値があります。関数空間表現へ移ることで、_無限個_ のパラメータをもつモデルを、有限の計算量で表現する方法を導出したのです。RBF カーネルをもつガウス過程は _万能近似器_ であり、任意の連続関数を任意の精度で表現できます。上の導出から、その理由は直感的にも分かります。$\ell \to 0$ として各放射基底関数を点質量に潰し、各点質量に好きな高さを与えればよいのです。

したがって、RBF カーネルをもつガウス過程は無限個のパラメータをもつモデルであり、どんな有限のニューラルネットワークよりもはるかに柔軟です。もしかすると、_過剰パラメータ化_ されたニューラルネットワークをめぐる大騒ぎは見当違いなのかもしれません。これから見るように、RBF カーネルをもつ GP は過学習せず、実際、小規模データセットでは特に優れた汎化性能を示します。さらに、 :cite:`zhang2021understanding` にある、ランダムラベル付き画像を完全に当てはめられるのに、構造化された問題では依然としてよく汎化する、といった例は、ガウス過程を用いて完全に再現できます :cite:`wilson2020bayesian`。ニューラルネットワークは、私たちが思っているほど別物ではないのです。

RBF カーネルをもつガウス過程や、_長さ尺度_ のようなハイパーパラメータについてさらに直感を深めるために、関数分布から直接サンプルしてみましょう。これまでと同様、手順は単純です。

1. GP に問い合わせたい入力 $x$ の点 $x_1,\dots,x_n$ を選ぶ。
2. $m(x_i)$ を $i = 1,\dots,n$ について、また $k(x_i,x_j)$ を $i,j = 1,\dots,n$ について評価し、それぞれ平均ベクトルと共分散行列 $\mu$ と $K$ を作る。ここで $(f(x_1),\dots,f(x_n)) \sim \mathcal{N}(\mu, K)$ である。
3. この多変量ガウス分布からサンプルして、サンプル関数値を得る。
4. さらに何度もサンプルして、それらの点で問い合わせた複数のサンプル関数を可視化する。

この過程を下の図に示します。

```{.python .input}
def rbfkernel(x1, x2, ls=4.):  #@save
    dist = distance_matrix(np.expand_dims(x1, 1), np.expand_dims(x2, 1))
    return np.exp(-(1. / ls / 2) * (dist ** 2))

x_points = np.linspace(0, 5, 50)
meanvec = np.zeros(len(x_points))
covmat = rbfkernel(x_points,x_points, 1)

prior_samples= np.random.multivariate_normal(meanvec, covmat, size=5);
d2l.plt.plot(x_points, prior_samples.T, alpha=0.5)
d2l.plt.show()
```

## ニューラルネットワークカーネル

機械学習におけるガウス過程の研究は、ニューラルネットワークの研究によって引き起こされました。ラドフォード・ニールは、ますます大きなベイズニューラルネットワークを追求し、最終的に 1994 年に（後に 1996 年に出版されました。これは最も悪名高い NeurIPS の却下例の 1 つだったためです）、無限個の隠れユニットをもつそのようなネットワークが、特定のカーネル関数をもつガウス過程になることを示しました :cite:`neal1996bayesian`。この導出への関心は再燃しており、ニューラル接線カーネルのような考え方が、ニューラルネットワークの汎化特性を調べるために使われています :cite:`matthews2018gaussian` :cite:`novak2018bayesian`。以下のようにしてニューラルネットワークカーネルを導出できます。

1 つの隠れ層をもつニューラルネットワーク関数 $f(x)$ を考えます。

$$f(x) = b + \sum_{i=1}^{J} v_i h(x; u_i).$$

$b$ はバイアス、$v_i$ は隠れ層から出力層への重み、$h$ は任意の有界な隠れユニットの活性化関数、$u_i$ は入力から隠れ層への重み、$J$ は隠れユニット数です。$b$ と $v_i$ は独立で、それぞれ平均 0、分散 $\sigma_b^2$ と $\sigma_v^2/J$ をもち、$u_i$ は独立同分布に従うとします。すると中心極限定理を用いて、任意の関数値の集合 $f(x_1),\dots,f(x_n)$ が同時多変量ガウス分布に従うことを示せます。

対応するガウス過程の平均関数と共分散関数は次のとおりです。

$$m(x) = E[f(x)] = 0$$

$$k(x,x') = \textrm{cov}[f(x),f(x')] = E[f(x)f(x')] = \sigma_b^2 + \frac{1}{J} \sum_{i=1}^{J} \sigma_v^2 E[h_i(x; u_i)h_i(x'; u_i)]$$

場合によっては、この共分散関数をほぼ閉じた形で評価できます。$h(x; u) = \textrm{erf}(u_0 + \sum_{j=1}^{P} u_j x_j)$、ここで $\textrm{erf}(z) = \frac{2}{\sqrt{\pi}} \int_{0}^{z} e^{-t^2} dt$、かつ $u \sim \mathcal{N}(0,\Sigma)$ とします。このとき $k(x,x') = \frac{2}{\pi} \textrm{sin}(\frac{2 \tilde{x}^{\top} \Sigma \tilde{x}'}{\sqrt{(1 + 2 \tilde{x}^{\top} \Sigma \tilde{x})(1 + 2 \tilde{x}'^{\top} \Sigma \tilde{x}')}})$ となります。

RBF カーネルは _定常_、つまり _平行移動不変_ であり、したがって $\tau = x-x'$ の関数として書けます。直感的には、定常性とは、入力空間を移動しても、変化率のような関数の高レベルな性質が変わらないことを意味します。一方、ニューラルネットワークカーネルは _非定常_ です。以下では、このカーネルをもつガウス過程からのサンプル関数を示します。原点付近で関数の見た目が質的に異なることが分かるでしょう。

## まとめ


ベイズ推論を行う最初のステップは、事前分布を指定することです。ガウス過程は、関数全体にわたる事前分布を指定するために使えます。従来の「重み空間」からのモデリングの見方から出発すると、モデルの関数形を先に定め、そのパラメータに分布を導入することで、関数に対する事前分布を誘導できます。あるいは、カーネルによって性質が制御される関数空間で、事前分布を直接指定することもできます。関数空間アプローチには多くの利点があります。実際には無限個のパラメータに対応するモデルを、有限の計算量で構築できるのです！ さらに、これらのモデルは非常に柔軟である一方、どのような関数が事前に起こりやすいかについて強い仮定を置くため、小規模データセットでは比較的良い汎化を示します。

関数空間におけるモデルの仮定は、直感的にはカーネルによって制御されます。カーネルはしばしば、滑らかさや周期性のような関数の高次の性質を符号化します。多くのカーネルは定常的で、つまり平行移動不変です。定常カーネルをもつガウス過程から引かれた関数は、入力空間のどこを見るかに関係なく、おおむね同じ高レベルの性質（たとえば変化率）をもちます。

ガウス過程は比較的一般的なモデルクラスであり、パラメータにガウス事前分布がある限り、多項式やフーリエ級数など、私たちがすでに知っている多くのモデル例を含みます。また、パラメータにガウス分布がなくても、無限個のパラメータをもつニューラルネットワークも含まれます。ラドフォード・ニールによって発見されたこのつながりは、機械学習研究者をニューラルネットワークからガウス過程へと向かわせるきっかけとなりました。
