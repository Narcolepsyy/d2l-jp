#  前提知識
:label:`chap_preliminaries`

ディープラーニングの世界に飛び込む準備として、
いくつかの基本的なスキルが必要になります。
(i) データを保存し操作するための技術；
(ii) さまざまなソースからデータを取り込み、
前処理するためのライブラリ；
(iii) 高次元データ要素に適用する
基本的な線形代数演算の知識；
(iv) 損失関数を減少させるために
各パラメータをどの方向に調整すべきかを
判断するのに十分な微積分の知識；
(v) 自動的に導関数を計算する能力。
これにより、今学んだ微積分の多くを
忘れてしまっても大丈夫です。
(vi) 不確実性の下で推論するための
私たちの主要な言語である確率論の
基本的な流暢さ；
ドキュメントから答えを見つけるための適性です。

要するに、この章では本書の
*大部分* の技術的内容を追っていくために
必要となる基礎を、手早く紹介します。
