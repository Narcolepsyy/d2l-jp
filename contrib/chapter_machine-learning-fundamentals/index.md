# 機械学習の基礎
:label:`chap_ml-fundamentals`

:numref:`chap_introduction` で示したように、
深層学習は機械学習の問題を解くための数ある一般的な手法の一つにすぎません。
線形回帰、ソフトマックス回帰、そして多層パーセプトロンの学習で見てきたように、
最適化アルゴリズムは
モデルパラメータを反復的に更新することで
損失関数の値を減少させます。
しかし、
深層ニューラルネットワークのような高容量モデルを学習するときには、
過学習のリスクがあります。
したがって、まずは
過学習、未学習、モデル選択という概念について、
厳密に導入する必要があります。
これらの問題に対処するために、
重み減衰やドロップアウトといった正則化手法を導入します。
機械学習の*導入*に失敗した事例が数多くあることを踏まえると、
いくつかの一般的な懸念を明らかにし、
こうした状況を早期に検出し、被害を軽減し、機械学習を責任ある形で利用するために必要な批判的思考を促すことが重要です。
全体を通して、概念だけでなく、
機械学習モデルを使う実践についても確かな理解を得られるようにすることを目指します。
この章の最後では、
ここまでに導入した内容を実際の事例、すなわち住宅価格予測に適用します。
モデルの計算性能、スケーラビリティ、効率性に関する事項は、後続の章に先送りします。
